Python and Visual Basic

These NEA resources are designed for Python or Visual Basic. Each resource will simply run from a folder, as no internet access is allowed during an Non Exam Assessment.

They are designed to run on any Web Browser (Windows or Mac)





How to Install

To install, extract the folder from your downloaded .zip file and drag the contents to a user area or network drive for students to access.
Use index.html to load the resource in a web browser.
